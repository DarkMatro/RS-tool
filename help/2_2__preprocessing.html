2.2. Preprocessing

Import
  • If the loaded spectra have different wavelength ranges, you will need to select the target range and click the Interpolate button. If all ranges are the same, the button will be unavailable.
  • Despike - noise removal (spikes).
  • The algorithm is implemented according to the article Iterative mollifier-based despiking of the single-scan Raman spectra
    Convert to 1/cm
 
  • Set the Laser wavelength,nm parameter equal to the laser wavelength. (in our case, 784.5 nm).
  • Max CCD value - the maximum intensity value captured by the camera.
  • Сick Convert - the Converted graph will show spectra converted to wave numbers.
    Limit range
  • The graph also has an area with two boundaries - these are the boundaries for cutting the spectrum. The boundary values ​​are specified in the parameters. The N points parameter affects the position of the automatically determined boundary. The higher the value, the more to the right the value in nm for the left boundary.
  • Click Cut spectrum - the graph will display the spectra in a reduced range.
    Baseline correction
  • Select the algorithm and parameters.
  • Click the Baseline correction button - the Baseline corrected graph will show spectra without a fluorescent background.
  • There are 42 different algorithms available.
  • If you click the '1' button at right bar and select a spectrum in the table, the calculated baseline will be displayed.
    Normalization
  • Select the algorithm and parameters.
  • Click the Normalization button
    Smoothing
    Select the algorithm and parameters.
    Click the Smooth button
    It is recommended to use MLESG or Savitsky-Golay filter. For example, we will use the standard MLESG settings with sigma=4. There are 14 different algorithms available.
    Limit range
  • Select final ranges and click Cut spectrum
    At this point, the pre-processing of the spectra ends and as a result we obtain a smoothed, normalized spectrum without a fluorescent component, i.e. a pure Raman spectrum.
     
    If you do not plan to subtract or separate lines further, then you need to go straight to the 4th tab Stat analysis and train the model based on the Baseline corrected or smoothed dataset. More details in Chapter 2.4.
     
    Clicking Average - Update updates the graph with the confidence interval.
