1.1. Меню

Меню File
 
New project - Создание пустого файла проекта в выбранном файле.
Open project - Открытие файла проекта через диалог выбора файла
Open recent - Открытие файла проекта через список последних открытых файлов (длина списка задается в настройках программы, по стандарту 10) (сам список хранится в файле recent-files.txt)
 
Import files - Загрузка спектров в программу (.txt, .asc). (поддерживаются любые текстовые форматы, где данные представлены двумерным массивом.)
 
Export - Files, nm - сохранение спектров в формате .asc в выбранной папке. Создается папка "nm" в ней сохраняются файлы. Сами спектры берутся с этапа 'Imported, nm'.
Export - Files, cm-1 - сохранение спектров в формате .asc в выбранной папке. Создается папка "cm" в ней сохраняются файлы. Сами спектры берутся с этапа после корректировки базовой линии.
Export - Averaged, cm-1 - сохранение спектров в формате .asc в выбранной папке. Создается папка "averaged" в ней сохраняются файлы. Берутся усредненные спектры каждой группы.
Tables to excel - сохранение данных всех таблиц в формате excel.
Production project - сохранение файла проекта, который предназначен только для классификации новых спектров. В нем недоступны изменения любых параметров.
Files nm to .csv - сохраняет исходные файлы в CSV таблицу
Decomposed lines to .csv - сохраняет таблицу с амплитудами разделенных линий в формат CSV.
 
Fit template - импорт / экспорт модели для разделения линий и всех связанных параметров
 
Save all - сохранение текущего файла проекта
Save as - сохранение проекта в выбранном файле
Close Project - закрытие проекта, очистка всех форм
Help - открытие данной справки.
 
Меню Edit
 
Undo - отменить последнее действие
Redo - Повторить последнее действие
Clear - очистить данные на определенном этапе обработки (действие необратимо)
 
Меню Preprocessing
Здесь поэтапно последовательно доступна предобработка спектров
 
Меню Stat analysis
Здесь запускается процесс обучения классификаторов
Fit - обучить модель которая выбрана в поле Current classificator на обучающей выборке которая указана в поле Dataset
PCA / PLS-DA - Понижение размерности
Refresh fit result plots - заполнить графики на вкладке Fit results результатами выбранной обученной модели (модель указывается в поле Current classificator)
Refresh SHAP - обновить рисунки на вкладке SHAP по выбранному классификатору. При условии что при обучении этой модели была активна галочка use SHAP.
