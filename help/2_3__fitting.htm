2.3. Fitting

  • После предобработки переходим на 2 вкладку Fitting. В левой панели вверху жмем на кнопку Edit template (над графиком должно появиться слово Template. В этом режиме можно изменять, добавлять, удалять линии, т.е. настраивать общую модель). Перед разделением линий всех спектров нужно создать общий для всех групп шаблон.
     
  • Добавление и удаление линий доступно только при редактировании шаблона (Название "Template" в заголовке графика означает режим шаблона). Можно выбрать усредненные линии групп и один усредненные по всем группам (Average). При выборе спектра из левой таблицы по Filename отобразится спектр КР, это будет уже не режим шаблона.
     
  • Шаблон можно создать вручную добавлением линий через кнопку Add или создать автоматически через кнопку Guess. В обоих случаях нужно будет выбрать тип линии из списка. Доступно 14 видов линий (рекомендуется использовать Gaussian, Split Gaussian, Voigt, Split Voigt).
     
  • Установим параметр Max peak HWHM = 12 см-1, остальные параметры рассчитались автоматически. Заполним таблицу SPLIT BY INTERVALS - жмем правой кнопкой мыши по таблице, выбираем Auto-search borders. Должны заполниться значений границ по которым будет делиться спектр на почти независимые области. Разделение нужно чтобы уменьшить количество операций раз в ~1000.
    В таблице не должны быть первый и последний минимум спектра. Для спектра на рисунке не должно быть значений ~ 360 и ~1780. Если добавить крайнюю точку то алгоритм будет пытаться искать линии за пределами спектра и скорее всего упадет с ошибкой.
     
  • Opt method - выбирает наименьших квадратов (он быстрее) Least-Squares, Trust Region Reflective method. Можно попробовать другие.
  • Guess method - Average анализ только по одному усредненному спектру быстро, но не точно. All - по всем спектрам - дольше, но точнее.
  •  Max noise level - уровень ниже которого амплитуда линии будет считаться шумом и ниже этого порога новые линии не будут добавляться.
  • После создания адекватной для все групп модели разделения линий - жмем на кнопку Batch, по ней будут разделены линий всех спектров, старые результаты разделения перезапишутся. Результат будет скомпонован в таблице Deconvoluted на странице 3.
     
  • Нажимаем кнопку Guess - выбираем например Gaussian. Ждем завершения анализа. Должны создаться линии как рисунке. Проверяем адекватность созданной модели. Возле границ, указанных в таблице SPLIT SPECTRUM может добавиться 2 линии по одной с каждом стороны от границы. Можно убрать одну из них лишнюю.
 
Так мы получили готовую модель. Применим ее на все спектры. Для этого нажимаем кнопку Batch и ждем применения модели ко всем спектрам. После завершения справа внизу будут отображены результаты аппроксимации, будет указана статистика по всем спектрам. Заполнится на следующей странице вкладка с таблицей Decomposed lines.
Если мы хотим чтобы учитывалась кроме информации об амплитуде разделенных линии еще и их положения максимумов x0, то жмем на кнопку include x0 info (повторного расчета не нужно)
